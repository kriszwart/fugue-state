## Audio Specifications

### For Best Performance:
- **Format**: MP3 (widely supported)
- **Bitrate**: 128kbps (good quality, small size)
- **Sample Rate**: 44.1kHz
- **Ambient tracks**: 30-60 seconds loop, seamless
- **SFX**: Short duration (0.5-2 seconds)

## Quick Start with Placeholder Sounds

If you don't have audio files yet, the system will gracefully handle missing files.
You can add files later and they'll work immediately.

## Example: Creating a Seamless Loop in Audacity (Free)

1. Download Audacity: https://www.audacityteam.org
2. Import your audio
3. Select 1-2 seconds at the end
4. Effects → Fade Out
5. Select 1-2 seconds at the beginning
6. Effects → Fade In
7. Export as MP3 (128kbps)
